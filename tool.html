 How to Use

      
    
        
  1. Auto-load: On startup, the tool automatically fetches projects.json from the same folder (requires a local server like Live Server).
    2. 
        
  2. Direct Save (recommended): Click "Pick projects.json" once to link the actual file. After that, "Save projects.json" writes directly into the file — no download needed.
    3. 
        
  3. Fallback: If your browser doesn't support direct file writing, the Save button will download the file instead. Replace the old projects.json with the downloaded one.
    4. 
        
  4. Add Projects: Fill in the form, upload the image, and click "Add Project". Then hit "Save projects.json" to update the file.
    5. 
        
  5. Images: Place new image files in the projects/ folder using the same filename shown in the form.
